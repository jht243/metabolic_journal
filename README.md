# The Metabolic Journal

Evidence-based metabolic health content site. Publishes long-form blog posts, SEO landing pages, and interactive health tools. Deployed on Render, backed by Supabase Postgres.

> **Live site:** [themetabolicjournal.com](https://themetabolicjournal.com)

---

## Architecture

```
┌────────────────────────────────────────────────────────────┐
│  Render Web — gunicorn server.py                           │
│   GET /           → home page                             │
│   GET /blog       → blog post index                       │
│   GET /briefing/<slug> → individual blog post             │
│   GET /tools      → tools index                           │
│   GET /assessment → health assessment quiz                │
│   POST /api/subscribe → Buttondown newsletter signup      │
│   GET /health     → status JSON                           │
└────────────────────────────────────────────────────────────┘
                         │
                         ▼
┌────────────────────────────────────────────────────────────┐
│  Supabase Postgres — data layer                            │
│   BlogPost         — published articles                    │
│   LandingPage      — generated SEO pages                  │
│   ConsultationBooking — intake form submissions            │
└────────────────────────────────────────────────────────────┘
```

**Static SEO pages** are generated by `scripts/generate_seo_pages.py` and served directly by Flask.

## Local Development

```bash
# 1. Install deps
python3.11 -m venv venv && source venv/bin/activate
pip install -r requirements.txt

# 2. Set env vars
cp .env.example .env
# Fill in DATABASE_URL, SUPABASE_URL, SUPABASE_KEY, etc.

# 3. Run dev server
flask --app server run --debug
```

## Key Files

| File / Directory | Role |
|-----------------|------|
| `server.py` | Flask app — all routes |
| `src/config.py` | Settings (Pydantic, reads from env) |
| `src/models.py` | SQLAlchemy models |
| `src/seo/` | SEO cluster topology and keyword research |
| `src/newsletter.py` | Buttondown / SendGrid email integration |
| `src/og_image.py` | Per-post OG card generator (Pillow) |
| `src/distribution/` | Google Indexing API + IndexNow clients |
| `templates/` | Jinja2 templates |
| `scripts/generate_seo_pages.py` | Static SEO page content |
| `scripts/indexnow_submit.py` | Batch-submit all URLs to IndexNow |

---

## Deployment (Render)

- **Web service:** `gunicorn server:app --bind 0.0.0.0:$PORT --workers 2 --timeout 120`
- **Health check:** `GET /health`
- **Environment variables:** `DATABASE_URL`, `SUPABASE_URL`, `SUPABASE_KEY`, `INDEXNOW_KEY`, `SENDGRID_API_KEY`, `OPENAI_API_KEY`
